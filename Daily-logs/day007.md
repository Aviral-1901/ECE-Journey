# Day 7 : Subtractor and 2's complement

## Two's complement
To get 2's complement of a number, invert the number and add 1. Using this we can get negative number. So lets say for 7(0111) -> Invert(1000) -> Add 1 (1001) which is -7 in 2's complement.                   Negative of a number means when it is added to it's original value the result becomes 0. In 4 bit 2's complement 7 = 0111 and -7 = 1001 because 0111 + 1001 = 10000, the extra carry bit is discarded leaving 0000.

## XOR as controllable inverter
When a number xor 0 then same number comes out and when a number xor 0 then the number inverts.                               
10 xor 11 = (01) ,    10 xor 00 = (10)

## Same circuit Adder/Subtractor
Because of how 2's complement works we can make adder and subtractor in the same circuit. We use a signal called sub which decides whether the circuit performs addition or subtraction. We perform B xor sub. If sub=0 the original number passes through unchanged and if sub=1 the bits invert. Now in full adder circuit we have three input bits a, b, cin so for the first bit of the number we keep cin=sub so for addition nothing changes as sub=0. But for sub=1 the xor inverts the number and adding 1 through cin gives the 2's complement and the same circuit handles the subtraction.
