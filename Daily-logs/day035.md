## Gate 10
Gate 10 was about recall of alu operations and testing of some of the edge cases of alu.

## SLT overflow
For SLT if a is smaller then we get result as 1 and 0 otherwise. To prevent overflow bugs the ALU compares the sign bits (MSB) first. If the signs are different the negative number is instantly declared smaller. If the signs are the same it performs subtraction and checks if the result is negative. slt treats numbers as signed numbers and there is another operation called SLTU for unsigned numbers.

## ALU Instructions
andi uses AND operation. SLT is its own R-type instruction and different from BLT.


## Key Insights
ALU must be combinational because registers captures values only at edges and before edge arrives the calculation should be completed and settled which can only hold if alu is combinational otherwise it would waste one cycle in alu only.
