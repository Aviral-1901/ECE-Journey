## Gate8
Building register_4 bit and checking if x0 is hardwired to ground. A register is uninitiliazed at the start of simulation and has unknown value given by the simulator. If x0 is hardwired to ground it should give 0 at the very start as well. So we checked for x0 at the start of simulation to test it. So x0 was checked at t=0 before doing anything else to ensure it is truly 0 from the very start.
