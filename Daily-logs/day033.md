## B-Type and J-Type Instruction
J-Type has bigger immediate value than B-Type because J-type is a jump instruction and may need to jump long addresses so J-type need to hold bigger immediate value allowing it to jump +/-1Mb of memory addresses. J-Type used for function calls while B-type for loops and branches.
