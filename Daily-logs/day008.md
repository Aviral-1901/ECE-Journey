# Day 8 : Majority Voter and Mux_4to1

## Topics I covered
Majority voter outputs 1 if two or more inputs are high. It is used in fault tolerance systems so that even if one of the system/module fails the system donot fail as a whole. Mux 4:1 is made using 3 mux_2:1. We have 2 mux_2:1 which selects the output among i0/i1 and i2/i3, for this select bit (s0) is used. Then we have another mux which selects which mux result to output using select bit(s1).
