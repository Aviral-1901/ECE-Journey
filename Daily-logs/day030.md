## S-Type clarity
S-type instruction is : sw rs2, imm(rs1) <br>
rs1 is base address register. rs2 is the register whose value gets stored into memory.


## B-Type
B-Type instruction : imm[12] imm[10:5] rs2   rs1   funct3 imm[4:1] imm[11] opcode <br>
0xFE628AE3 -> 1 111111 00110 00101 000 1010 1 1100011 <br>
OPCODE = 1100011 <br>
funct3 = 000 <br>
rs1 = 00101 <br>
rs2 = 00110 <br>
imm[12] = 1 <br>
imm[10:5] = 111111 <br>
imm[4:1] = 1010 <br>
imm[11] = 1 <br>
imm[0] = 0 always <br>
immediate (binary) = 1111111110100<br>
