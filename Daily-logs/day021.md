## Parking gate fsm
IDLE:    car_present=1 -> WAITING [open=0, alarm=0]<br>
IDLE:    car_present=0 -> IDLE    [open=0, alarm=0]<br>
WAITING: ticket_valid=1 -> OPEN [open=1, alarm=0]<br>
WAITING: ticket_valid=0 && counter==10 -> ALARM [open=0, alarm=0]<br>
WAITING: ticket_valid=0 && counter<10 -> WAITING [open=0, alarm=0]<br>
OPEN:    counter==5 -> IDLE [open=0, alarm=0]<br>
OPEN:    counter<5 -> OPEN [open=1, alarm=0]<br>
ALARM:   counter==5 -> WAITING [open=0, alarm=0]<br>
ALARM:   counter<5 -> ALARM [open=0, alarm=1]<br>

This parking gate fsm has 4 states. Gate opens for 5 seconds if ticket_valid and if ticket not valid and counter has passed 10 cycles(seconds) without valid ticket then it goes to alarm state where the alarm rigns for 5 seconds.
